# English

All content must be in English:

- Source code (identifiers, comments, docstrings)
- Commit messages and branch names
- Documentation in `docs/`, `CLAUDE.md`, and `.claude/`
- Rule and skill files

Exception: string literals and values that represent domain-specific regulatory or business data may remain in their original language when required by the product domain.
